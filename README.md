# 中醫嘉年華｜體質探索站 V8.1 正式版

## GitHub Pages 上傳方式
請將以下內容全部上傳到同一個 repository 的根目錄：

- `index.html`
- `style.css`
- `script.js`
- `assets/` 整個資料夾裡面的所有圖片

GitHub Pages 的圖片路徑已設定為 `assets/檔名`，請不要把圖片放成 `assets/assets/檔名`。

## 功能
- 手機版封面海報入口
- 性別選擇
- 第 1 階段 16 題初步評估
- 第 2 階段體質問卷
- 同分時支援多份第二階段問卷
- 結果頁支援多體質顯示
- 儲存結果 PNG

## 注意
本測驗為健康促進與衛教參考，非醫療診斷。


## V8.1 更新
- 第一階段改為：同一體質兩題合計 ≥5 分才進入第二階段。
- 若第一階段全部體質皆 <5 分，直接進入平和體質問卷。
- 平和體質維持反向計分，60 分以上判定為「是」。
- 統一問卷與結果頁圖片展示尺寸。
- 封面使用底部留白版本，按鈕由網頁呈現。
